#34 你需要一台内存在1G以上的vps,为什么要vps？因为国内的下载资源被共匪墙掉了，国内的网络无法搭建好完整的编译环境。vps需要安装centos7或者Linux内核在3.10以上的OS,这是docker虚拟机安装的最低要求。


命令行如下:
wget https://storage.googleapis.com/golang/go1.4.2.linux-amd64.tar.gz
tar -xvf go1.4.2.linux-amd64.tar.gz
sudo cp -r go /usr/local/go   

source /etc/profile

sudo yum update
curl -O -sSL https://get.docker.com/rpm/1.7.0/centos-6/RPMS/x86_64/docker-engine-1.7.0-1.el6.x86_64.rpm                 

sudo yum localinstall --nogpgcheck docker-engine-1.7.0-1.el6.x86_64.rpm
sudo service docker start

git clone -b valencia --depth 1 --single-branch git://github.com/getlantern/lantern.git
cd lantern
make docker
等二十分钟左右，编译环境创建完毕

然后编译源代码

make binaries
