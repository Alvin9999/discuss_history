#6 xxnet....@gmail.com 
因为开发手机，中断了近1个月更新XX-Net，积累了好几个Bug，还有好几个提交的补丁。
看样子手机开发不是那么容易，所以这周花了几天，把欠各位的先补上，发个新版： 1.16.0

主要更新内容：
+ 增加了ip评估机制
  策略是按SSL握手时间进行排序，前面第100个ip的握手时间作为标准，
  动态调整扫描ip的线程数。
  
  新下载的用户，会开足马力扫描，等扫描足够多足够好的ip之后，就会把线程降下来，减少资源消耗。
  当然你不想自动调整也可以。

+ Windows 托盘菜单，显示当前的代理状态
  感谢 R Shishioh的补丁，已经调整合并进去

+ 修复部署7个appid后卡住
  原因是控制台输出没有被处理，导致阻塞

+ 修复新版Linux下，自动升级时不接受GoAgent证书
  

请大家帮忙测试，有问题我尽快修复
