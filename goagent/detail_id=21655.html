P2P 网络, 任何人都可以在这个网络里发布网站，由于不需要服务器，建站显然是免费的。

新手想建站但不会写网站代码的话可以用作者写好的模板 https://github.com/HelloZeroNet ，可用来建博客，讨论站等。作者还计划发布去中心化，不怕服务关闭的 Github 和 Google+ 。

这个源码有看过，存在几个问题：
1. 目前启动依赖几个中心节点，容易封锁
2. bt 通讯部分，容易泄漏ip，被定位到
3. 还处于试验阶段，没有任何保护措施，容易被挂马、钓鱼

这类项目可以作为辅助或则研究，目前还不适合作为一个长期的交流平台
