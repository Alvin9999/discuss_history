Issue 22222: 大半夜的码了好多字，你有耐心看完吗？
3 people starred this issue and may be notified of changes. Back to list
Status:  New
Owner:  ----
Type-Defect
Priority-Medium


 
Reported by allgg.u...@gmail.com, Aug 5, 2015 
眼看着时间一天天的逼近，距离Google Code 关闭也只有20天的时间了。

相信有相当一部分人抱着不舍或者无奈的心情。以后这里就是考古区了，逐渐成为历史。

有人建议开论坛，集合大众，当然也看到某人费力的在宣传自己的扣扣群、Google Group，被删后接着发，一遍一遍，似XX的步伐。

首先说下，论坛的建立，要有良好的空间、域名，后期的维护、管理。

域名可以免费或付费。

但空间的要求较多，服务器的地点选址，性能，速度等等约束着，

其次，空间免费与收费。

免费的容易宕掉，或者在线时间不稳定。

收费的相对稳定，能保证在线时间，墙与被墙其实都是次要的了，但谁会去建立一个树大招风的网站

网络上有，凤毛麟角，但本着事不关己的心态，我想很多人还是愿意跟风在后面，所谓枪打出头鸟

更何况后期的维护、管理，所以可以想象到GoA作者的压力也是巨大的。

就象在某贴中看到的，既然翻墙出来了，免不了看看政治上的东西，或阳光或阴暗，但看看兴许没问题。一旦发表，或激进或反动，后果可想而知。

请千万不要小视我国的大网络COP，权限很大的。

大家都是成年人，不会想当然的，当然也不排除某些人物想当然的抱着这个世界真的是骗子都不够用了傻子太多的想法。

所以基本可以Pass掉这个方案。既自费或免费建立一个有序或无序的论坛。

那么问题来了。时间逼近。

何去何从 V 1.0

大家都是希望能组团的来讨论，或墙或政治，或Adult Tube -_-!!

但是基于软件的issues我想还是大部分人关心的。

比如如何快速的观看Tube，怎样减少重复查找IP的劳动，怎样保护自己的隐私等等。

我的想法是要么Google Group，要么Github，既作者新版本的GoProxy GitHub Issues

Google Group作者没有建立，第三方的相信很多人也不敢轻易靠近。

好处在于都是Google旗下的，稳定，而且脸熟，既ID大家大体都相识，

但group的特点既论坛的特点，默认按日期排序。新帖子排在考前，是否有按回复排序这个功能？我没深入研究，只是看了个大概。

而这个Issues，默认按回复排序，也可以自定义Sort By ID，当然这都已经不重要了。

作者近期在开发新的软件，相信大家都很期待。

当然也提供了讨论平台。

就目前而言，Github还没有被墙，也不可预见

有的人讲Github是程序员的家，当初被墙着实让很多人苦恼，也有网络大V来呼吁，后来才解除封禁。

很多人担心GFW本着宁可错封一千也不放过一个的态度，再次让这个基地面临被封的境地。

因为未知，所以只能得过且过。

我不知道这个帖子是否有倾向引导大家去Github，但是我只是说出自己的想法。

另外Github上的Issues可以排序哦:)

我知道的是我看到的新闻，或阳光或阴暗的，只有我自己知道

我也帮助过我的朋友去翻墙，因为他需要访问Google，很好奇YouTube是什么东西

国内youku都在转载着国外的，好似前阵子的E3，我基本看到喜欢的都传到自己的优酷上，有人就很好奇问我从哪里找到的，我回答曰：U2B

综上所述我想说，你改变不了的，那就只能随大流，默默的为你自己活着

前日跟朋友说起墙的事，我想着要不要教教他，他很不屑。

技术这种东西，为己所用已足够了，能相对自由的畅游网络已经很不易了。

有需要的人你能帮助到他也很好，但不要强扭。因为不是每个人都关心。

所以Google Code关闭不是什么惊天动地的大事。Google也要运营的。

貌似也没啥结论，大家如果抱着比翻的决心，相信网络里总会有你需要的资料的。

兴许哪天你自己捣鼓出个小软件顺墙翻出来了呢

也很感谢作者辛苦的劳作，此Issues能有20k的条目也是日积月累出来的。

新版本的软件很期待，我也会继续支持的。

何去何从 V 2.0

相信大家自己都有自己的想法的。今天的离别兴许是为了以后更好的重逢呢？

保不齐哪天没这墙了，人们还会念叨当年的翻墙史吗？

没有字体大小编辑，没有任何装饰，Google果然够简洁

好长，有耐心看完吗？
