#6 warmingf...@gmail.com 
用文本编辑器正则表达式的方式替换，以将 1.255.33.83|1.255.33.46|1.255.33.41|1.255.33.27 变成 

"1.255.33.83",
"1.255.33.46",
"1.255.33.41",
"1.255.33.27"

为例：

第一步： 替换 "|" 为 "," ： （匹配） \| （提换为） ,\n
第二步： 给 IP 地址们加上引号：（匹配） ((?:\d+\.?){4})  （替换为） "$1"

文本编辑器使用方法参考此帖： https://code.google.com/p/goagent/issues/detail?id=21315

#7 xyuanmu 
其实并不需要换行的，换行只是增加可读性，替换可以直接将 | 替换成 ", " ，然后首尾再添加英文冒号 " 即可，任何文本编辑器都能做到，个人认为逗号后留一个空格看起来不会太拥挤。

例如下面这些ip
74.125.106.90|211.124.13.179|74.125.106.70|61.87.203.33|49.98.25.107|103.16.204.97

写入iplist.json
{
	"iplist" : {
		"google_hk": [
			"74.125.106.90", "211.124.13.179", "74.125.106.70", "61.87.203.33", "49.98.25.107", "103.16.204.97"
		]
	}
}
