#2 rongw...@gmail.com 
psiphon 3  退出后,无法上网,解决方法

1. 关闭赛风后浏览器无法上网”的原因，或许是你没正确关闭所致

   正常方式 : 先点击 "绿色勾号" 变更为"灰色勾号" ,再关闭"赛风网页"

1. IE -> [ internet 选项 ] -> [ 连接 ] -> 点击 [ 宽带连接 ] 的 [ 设置 ]

   检查 [ 代理服务器 ] 项 -> 去除勾选 [ 对此连接使用代理服务器 ]

2. 即使,原来[ 对此连接使用代理服务器 ] 没有勾选,

   也必须,重新 按 [确定 ] ,以便重置为,没有使用代理服务器的状态
