#4 xyuanmu 
花点时间码字，gae模式使用方法：
1. 下载解压后用记事本打开：gae.json，推荐用Notepad2打开；
2. 在 "appids" 下一行的 "goagenta" 修改为自己的appid，格式："appid1","appid2","appid3" 全部符号都是半角，而且最后一个id后面没有半角逗号，切记；
3. 打开 iplist.json，在 google_cn 和 google_hk 下方添加自己的ip地址，否则可能无法使用，方法同上；
4. 打开 main.json，找到 "gae", 并将这一行移动到 "php", 上，效果如下所示：
		"roundtrip": [
			"autoproxy",
			"auth",
			// "iplist",
			"gae",
			"php",
			"direct"
		],
5. 自己手动导入 GoAgent.crt 证书。

最后双击 goagent.exe 运行即可，端口由原来的 8087 改为 8000，可以自己修改 main.json 更换端口。
