Proxifier 占用资源高的解决方案
=======================

Proxifier 占用资源高，引起系统卡顿有可能是因为频繁输出日志。

解决这个问题有两点需要做

1. Proxifier 内设置，选中 Log -> Output Level -> Errors Only ，如图 https://i.imgur.com/WKrkR0s.jpg

这样做的理由： 选中 Errors Only 减少日志输出

2. 关掉 handle direct connections ，即去掉下图中 handle direct connections 前面的“对号” https://i.imgur.com/nfXu2ab.jpg

这样做的理由： 减少 Errors Only 类型日志输出（如扫描 Google IP 时会产生大量的连接出错的日志），如果你的红色错误日志不多的话，不关掉 handle direct connections 也行。
