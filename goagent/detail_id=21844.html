赛风的简单步骤：

第一步： 利用 Proxifier 使赛风只能访问 yryirjrhqffsxwpg-a.akamaihd.net 这个 Akamai CDN 服务，不然赛风有可能自己选择不使用 Akamai CDN 加速
第二步： 修改本机 hosts ，打开 "C:\Windows\System32\drivers\etc\hosts" 添加下面一行后保存在

210.0.156.15 yryirjrhqffsxwpg-a.akamaihd.net

（210.0.156.0/24 这组 Akamai IP 地址电信和移动访问都能极速，但是移动还有更好的选择，更多的 Akamai IP 地址可用 checkgoogleip 扫描） 
第三部： 设置赛风用新加坡服务器


详细步骤： 

1. 安装 Proxifier 这款软件（最好用 3.21 版，最新的 3.28 版对赛风支持不是很好）

2. 在 Proxifier 中如图 https://i.imgur.com/gfnYeKz.jpg 添加规则使赛风的进程 psiphon-tunnel-core.exe 只能访问 yryirjrhqffsxwpg-a.akamaihd.net

3. 使用 hosts, dnsforwarder, 3proxy 任其一使 yryirjrhqffsxwpg-a.akamaihd.net 定向到一个香港的 Akamai CDN 节点（如 210.0.156.15 ），这里推荐修改系统 hosts

4. 设置赛风使用新加坡服务器，香港访问她很快。

看了那么多，给出个更有效且简单的加速赛风方法，成功几率大，如果你的带宽有100M的话，简直是8k youtube一路顺风：

1，此方法是基于楼主及上面几位大神提供的思路而成的，就此多谢他们。

通过防火墙规则定制psiphon-tunnel-core.exe的进出，只允许psiphon-tunnel-core.exe连接指定的
prfzquobemaspprl-a.akamaihd.net
alchzwwfzmrtgoaf-a.akamaihd.net
jkiysqntxacscicm-a.akamaihd.net
yryirjrhqffsxwpg-a.akamaihd.net

设定的规则一：指定四个之一或四个都同时允许，规则二：阻止所有连接，而且使得规则一优于规则二方能生效，目的就是让psiphon-tunnel-core.exe此进程只连接上述四个域名之一或全部四个。

然后找到适合自己的Akamai CDN ip，通过hosts指向上述四个节点之一或全部四个。

xxx.xxx.xxx.xxx prfzquobemaspprl-a.akamaihd.net
xxx.xxx.xxx.xxx alchzwwfzmrtgoaf-a.akamaihd.net
xxx.xxx.xxx.xxx jkiysqntxacscicm-a.akamaihd.net
xxx.xxx.xxx.xxx yryirjrhqffsxwpg-a.akamaihd.net

设定完hosts后记得刷新dns，通过cmd: ipconfig /flushdns 才能生效，再不成重启一下计算机。

如果你装的防火墙是comodo，还可以通过显示连接实时知道连接的akamai ip，从而知道所设置是否成功。

声明：此法只适合已经安装有防火墙，而且该防火墙的规则定义支持域名定义（好像windows自带的防火墙不支持，或者我找不到定义的位置，请见谅或者请大神指教方法），而你又不想使用Proxifier（始终要多安装一个软件且多耗资源），那么此法将可以供你使用。
