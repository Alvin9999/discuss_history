#18 kingtope...@gmail.com 
#17 你有所不知，lantern编译不是你想的那么简单的，不是简单的把源码下载下来编译，而是要把整个git repo clone下来，然后选取特定版本的分支才能编译。 你直接下载源码，根本无法编译

我是这样操作的：

git clone git://github.com/getlantern/lantern
cd lantern
git checkout 2.0.0-beta7 
make binaries

上面的前提是要事先配好golang1.4.2和docker的编译环境（docker会创建一个fedora22的虚拟机环境）

How to compile binaries from lantern source code:

Procedures:

wget https://storage.googleapis.com/golang/go1.4.2.linux-amd64.tar.gz
tar -xvf go1.4.2.linux-amd64.tar.gz
sudo cp -r go /usr/local/go   
 
vi /etc/profile
 
export GOROOT=/usr/local/go
export PATH=$PATH:$GOROOT/bin
 
source /etc/profile

sudo yum update
curl -O -sSL https://get.docker.com/rpm/1.7.0/centos-6/RPMS/x86_64/docker-engine-1.7.0-1.el6.x86_64.rpm

sudo yum localinstall --nogpgcheck docker-engine-1.7.0-1.el6.x86_64.rpm
sudo service docker start

git clone -b valencia --depth 1 --single-branch git://github.com/getlantern/lantern.git
cd lantern
make docker
(will wait a long time for fedora22 environment buiding)

make binaries

Finished! 
And then transfer lantern binaries to your local computer by FTP
