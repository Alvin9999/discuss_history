Reported by u78...@gmail.com, Jul 13, 2015 

win7 ie8 goagent 3.2.3大家都说goagent 3.2.3不能使用前置代理。要用goagent 3.1.9或者3.2.0 r42版本。但我就直接用新版，因为新版可以用gvs的ip。

meow 1.2版本。地址：https://github.com/renzhn/MEOW

shadowsocks的账号假设：proxy = ss://rc4-md5:密码@1.2.3.4:8388 假设是日本的shadowsocks

1.meow的设置：就是把rc.txt里面的某些#去掉，把shadowsocks下面那行的#去掉并换成你的shadowsocks服务器地址，密码，端口，加密方式。这里假设是日本的shadowsocks，加密方式是rc4-md5 
# shadowsocks:
   proxy = ss://rc4-md5:密码@1.2.3.4:8388
这时候访问speedtest.net显示日本。说明正确。

2.meow的设置：就是把rc.txt里面的某些#去掉，把http那行的#去掉并把端口改成goagent的8087，把shadowsocks下面那行的#去掉并换成你的shadowsocks服务器地址，密码，端口，加密方式。这里假设是日本的shadowsocks，加密方式是rc4-md5 
    proxy = http://127.0.0.1:8087
#   proxy = http://user:password@127.0.0.1:8080
# shadowsocks:
    proxy = ss://encrypt_method:password@1.2.3.4:8388
然后修改goagent文件夹下面local文件夹下面的proxy.ini里面的[proxy]字段，修改如下
[proxy]
enable = 1
autodetect = 1
host = 127.0.0.1
port = 4411
username =
password =
最后用goagent设置IE的代理为127.0.0.1:8087，访问speedtest.net显示google cloud
是不是说明前置代理设置成功？

我也搞糊涂了 不知道meow goagent shadowsocks怎么配合。希望大神指教！谢谢！

#1 redfrog...@gmail.com 
除非你是极端恶劣的网络环境，否则你这么设置得不偿失，ss的流量太小了。
更何况ss的速度不见得比goa好。
我这里是因为网络环境太差，lag严重采用ss为前端降低lag为goa提速的。
你如果网络环境ok，完全不用这么折腾。
