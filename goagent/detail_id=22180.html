#10 kingtope...@gmail.com 
#9 如果你是linux小白也没有关系，我也是从这个阶段过来的，到现在为止我还是自认为自己仍然是linux菜鸟。

你首先要学会用putty使用SSH连接到VPS，这个很简单的，我就不说了，你自己google一下。

putty使用的关键点，直接点击右键等于粘贴，切记，这是个非常重要的技巧！

通过putty的SSH远程连接到VPS以后，你会看到一个黑屏的终端界面。

运行下面的命令就可以升级了。

pip install -U shadowsocks && reboot

重启登录看看版本号： ssserver --version 或者  sslocal --version
