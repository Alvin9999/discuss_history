#11 Kisesy 
我说下改法,其他平台也许能用,因为并没有改动程序代码,而是对程序的执行思维进行改动,我是在win上改的,其他系统不了解

1. 便携
蓝灯的设置保存到%AppData%路径,也就是 C:\Users\username\AppData\Roaming
那么蓝灯怎么知道这个路径呢? 方法是读系统变量,读到了就保存在AppData
如果读不到,按照windows的文件处理机制,它就会把文件存到当前目录,那么就把 APPDATA 替换为空就行
## 用十六进制编辑器把APPDATA替换为空字符

2. 去弹窗

源代码中有这么一段
if externalUrl != "NO"+"_URL" && !openedExternal {
			time.Sleep(4 * time.Second)
			open.Run(externalUrl)
			openedExternal = true
		}
看到这里有个判断,那就好办了,现在的externalUrl是https://www.facebook.com/manototv/
## 那么只需要用十六进制编辑器把 https://www.facebook.com/manototv/ 替换为NO_URL就行了


关于为什么不去除 localhost:16823 弹窗
是因为不太好去,因为看代码,弹窗是在函数里,那么程序启动时,会调用,而右键打开设置时也会调用
那么去掉了,设置就出不来了,又没法判断程序是刚打开还是右键打开的
我之前改过启动时就一个托盘,就是不能用localhost:16823的设置,其他功能都能用,但没设置用起来还是不太方便,所以我就没去
