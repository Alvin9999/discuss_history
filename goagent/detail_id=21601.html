#12 kingtope...@gmail.com 
我说下简要步骤:
1.安装goLang,配好环境变量
2.安装docker
3.从蓝灯的github上git clone repo，然后 cd lantern,再git checkout要编译的版本
4.make docker（创建编译环境需要的docker image虚拟机，只做一次，以后都不用做了）
5. 编译二进制文件：make windows or make Linux-amd64，很快就能搞好。
6.搭建ftp服务端，并把编译好的文件在本机从vps上下载下来
