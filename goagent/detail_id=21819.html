#5 kingtope...@gmail.com 
这个shadowsocks跟之前安装的版本不一样，是两套班子，需要重新安装服务端,才能使用“UDP over UDP和TCP混淆协议头” ，具体步骤如下：

1. VPS卸载老的shadowsocks
    pip uninstall shadowsocks

2. 安装新shadowsocks需要的依赖库：
centos：
yum install python-setuptools && easy_install pip
yum install m2crypto git

ubuntu/debian：
apt-get install python-pip
apt-get install m2crypto git

pip install cymysql

3.下载新的shadowsocks的源代码：
git clone -b master https://github.com/breakwa11/shadowsocks.git

4. 安装shadowsocks
   cd shadowsocks
   python setup.py install

5. 开启服务端（原先的/etc/shadowsocks.json文件不用修改）：
   ssserver -c /etc/shadowsocks.json -d start

6. 运行shadowsocks客户端，勾选UDP over UDP和TCP混淆协议头。 
   完成，enjoy!
