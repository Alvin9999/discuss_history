# 并不全然认同西方现行制度
从未觉得现今欧美国家实行的民主制度是足够理想的制度，相反其很多弊病也几乎是天然的，包括：大资本家代言人（从参议员到总统，最初的候选者绝对不是该民族的"圣人大贤"，亢奋的选民实际上多半只是做二三选一的题目而已）；政党利益交换（实际执政过程中，各种金融集团、行业集团的政策游说渗透等）；经济危机；少节制的商业化资本化造成短视、伤害社会公平公义（如在教育资源中，环境气候问题中）。

# 为什么民主制度在有的亚非拉国家没能也长成参天大树？
对这个问，觉得不能图方便，轻易归咎到制度本身上去。如果人类学没有弄错，那么人性这一最根本的水土就不会是霄壤之别，老马最关注的生产力也早已化为各国竞逐的科学技术这个普世圣杯，而种族、文化、历史这些虽然是影响政治制度的非常重要的要素，但终究也只是人性这块土壤的微量元素，政治的本质乃人性的化合作用的合集。虽然一样的种子不是在谁家都能保证结出又大又甜的西瓜，但要是没有好的种子，不论哪家的土壤种多少代都只能是歪瓜裂枣这个地球人都知道。事实上地球人也不能否认它取得的的巨大成功。

# 那，会不会是不是中国的社会制度其实是设计好了的，只是落实落了个空？
关于这个问题的另一个Beta版：金三胖不是也挺直腰杆高喊朝鲜民主主义人民共和国么？常常奇怪，不是无论从理论到实际，社会主义不是本质都更民主么？无论共产党所宣扬的中国特色到底被人民日报高大上到什么地步，被绕到哪一条沟沟弯弯里去了，可是要是缺乏那些政治进化的要义，首先就觉得根本没有落实的可能。

觉得根本的是：中国其实并没有找到自己的政治出路。因为本质就是相对立的概念，民权、民主当然不会被当局容忍，纵横五千年不都是在干着这件事吗，当前的事没任何费解的。所谓"摸着石头过河"其实远不光是要掰几块石头，整个河床都有待去挖掘，不然迟早被河水翻覆。
比起愤怒或冷漠，这个国家远更需要的是创造，悲哀的是，在当下的中国，就是这个创造的前提都没有打开。

七不讲

七不讲是中国共产党提出的“七个不要讲”内容的简称，首先由中国华东政法大学教师张雪忠在新浪微博上提及，随后得到其他学者和教授的证实[1]。该指示是由相关领导在开会时口述通知，可能会得到长期执行[2]。“七不讲”随后在新浪微博遭屏蔽[2]。

背景

2013年5月13日，中共中央办公厅印发了非公开发表的《关于当前意识形态领域情况的通报》，并下发到县团级供相关干部学习。通报提到“要切实加强对当前意识形态工作的领导，把加强意识形态领域的工作列入重要议事日程，做到有载体、有活动，形成学习制度”，要求相关官员同“危险的”西方价值观作斗争。涉及该通报的互联网内容都被删除或封禁[3][4]。

而最先揭露“七不讲”的张雪忠曾在2012年9月因发表对香港国民教育的看法，遭华东政法大学取消其对本科生的授课资格。他还曾致信中国教育部长袁贵仁，要求将马克思主义、毛泽东思想，邓小平理论等课程从大学公共课中去除[5]。

后续事件

张雪忠在新浪微博发表“七不讲”后其账号后遭到删除[5]。“七不讲”也成了新浪微博的搜索敏感词[7]。
各方反应

    曾任中共前总书记赵紫阳的政治秘书的鲍彤表示不能确定“七不讲”内容是否属实，如果是真的，习近平提出的“中国梦”将一夜回到“辛亥革命”之前的“皇帝梦”。若不是事实，《人民日报》和新华社应宣布这是谣言，若不进行辟谣，又在高校当中流传，这就说明主旋律混乱。[1]
    历史学者章立凡称中共当局喜欢将口号“数字化”，如“四项基本原则”、“五不搞”、 “三个自信”等。这些口号也体现出中国共产党一成不变的执政思维，这种僵化的统治必将引发自下而上的社会变革。章立凡认为在互联网时代，因为人们的思想没那么容易被钳制，“七不讲”不会给执政者带来期盼的效力。[1]他表示“七不讲”其实是“两个凡是”的发展，当局推出七个不能触及的禁区，实际上反而是在提醒大家，这是中国现今体制上的七个关键弊端。“两个凡是”即为“凡是毛泽东说的都是对的，凡是毛泽东的指示必须坚决执行”。报道说，2013年是毛泽东冥诞120年，中国当局近来多次收紧言论被指与此有关。[2]
    民主党人士查建国称“七不讲”的内容非常荒谬可笑，若此事属实，必将遭高校知识分子群起反抗，他说“我怀疑这种消息的真实性，因为七不讲的内容太惊人、太可笑了，而且不可能执行。若是真的，必会遭到抵制，遭到大规模的反对，这是一个很大的倒退。”[8]
    北京学者莫之许说：“以前虽然没有明文规定，但他们的教学方针就是这样子，因为没有明文规定，有些老师讲这些东西不太会受到惩罚。但现在这样规定的话，若有教师讲这些东西的话就会被处罚、不让上课等等。”[8]
    北京理工大学教授胡星斗认为中共当局“一切工作的出发点都是为了维稳”，估计是由于现在社会形势非常严峻，中国百姓不满的声音非常大，所以向大学传达“7个不要讲”精神，“可是这样也就等于走进了死胡同”。[9]他表示也许是中共感到社会危机愈来愈严重，所以要从意识形态方面维稳，“但我认为这样只会陷入更大的危机”。[2]
    法律学者徐昕引述网民称：“民主是一种很复杂的东西，复杂到这是中国人唯一没能山寨成功的东西。”[2]
    中国知名地产商任志强听闻后称：“要把权力关进笼子，就不能没有新闻自由”。[9]
