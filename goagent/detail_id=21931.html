#5 yredcarp...@gmail.com 
#4 首先要有ss账号
http://ss.st2201.com/user/reg.php
修改 rc.txt 文件，加一行
proxy = ss://encrypt_method:password@x.x.x.x:xxxx
把上面一行中的encrypt_method改为你的ss的加密方式，同样，password改为ss的密码，x.x.x.x改为ss的服务ip，xxxx改为ss提供的端口号，注意： @之类的符号不要动

修改 goagent 的设置文件  proxy.ini 或 proxy.user.ini
[proxy]
enable = 1
autodetect = 1
host = 127.0.0.1
port = 4411
这已经够直白了吧
