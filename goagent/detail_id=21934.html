用SS做meow的前置代理，简单写下

下载meow
https://github.com/renzhn/MEOW

修改 rc.txt 文件，加一行

proxy = ss://encrypt_method:password@x.x.x.x:xxxx

具体的加密方式、服务器地址、密码、端口根据你实际内容来改

修改 goagent 的设置文件  proxy.ini 或 proxy.user.ini

[proxy]
enable = 1
autodetect = 1
host = 127.0.0.1
port = 4411

然后开着meow和goagent，浏览器里的代理设置方法还是用goagent的

完事
