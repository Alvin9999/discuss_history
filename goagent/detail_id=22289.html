#9  我只说一个最典型的torrc配置如下：


SocksListenAddress 127.0.0.1

Socks5Proxy 127.0.0.1:1080
UseBridges 0

ClientOnly 1
DataDirectory ./Data
GeoIPFile ./Data/geoip
GeoIPv6File ./Data/geoip6
Log notice file ./tor.log

注意：torrc一定不能是txt文件！ 从其他地方拷贝一个非txt的torrc来修改。

#20 ghtgftgj...@gmail.com 
我研究明白了

去官网下载，Expert Bundle，目前版本tor-win32-0.2.6.10.zip
https://www.torproject.org/download/download.html.en

建立一个文件夹tor123，把tor-0.2.6.10解压缩到tor123文件夹内。
tor123文件夹内现在有2个文件夹了，一个是Tor一个是Data
打开tor文件夹，在里面创建一个批处理，内容是tor -f torrc，百度搜索 如何创建批处理
在从其他地方复制一个torrc到tor文件夹里面。

torrc内容如下
#127.0.0.0如果改成0.0.0.0，可以监听虚拟机
SocksListenAddress 127.0.0.1

#端口 socks 9150
ControlPort 9151
SocksPort 9150

#下面是蓝灯前置代理，如是SS前置Socks5Proxy 127.0.0.1:1080，注意注意前缀是不一样的
HTTPSProxy 127.0.0.1:8787
UseBridges 0

ClientOnly 1
DataDirectory ./Data
GeoIPFile ./Data/geoip
GeoIPv6File ./Data/geoip6
Log notice file ./tor.log

===========================
第一次连接比较慢，经常出现60000ms
可以退出dos窗口，在任务管理器里在结束tor
之前几位大哥的，都是用SS前置代理，换成灯灯，Socks5Proxy得换成HTTPSProxy

启动批处理文件bat,内容是：tor -f torrc，启动就可以了，注意看tor.log,到了"Bootstrapped 100%: Done"就表示tor链路连接完全成功，浏览器socks设置127.0.0.1:9150

## Entry policies to allow/deny SOCKS requests based on IP address.
