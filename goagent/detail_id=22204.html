#11 Kisesy 
刚才停电了......
我总结一下修改的地方
只改 flightlight.go 文件

go func() {
    err := client.ListenAndServe(func() {
        pacOn()
        addExitFunc(pacOff)
        if showui && !*startup {
            ui.Show()
        } else {
            log.Debugf("Not opening browser. Startup is: %v", *startup)
        }
    })
    if err != nil {
        exit(fmt.Errorf("Error calling listen and serve: %v", err))
    }
}()

这是个关键函数:
把 pacOn()注释掉 可以去除设置pac
把 if showui && !*startup { 改为 if false { 可以去除启动弹窗
这两个功能只改这两个地方就可以,不用再改其他地方

我来说明一下蓝灯的版本问题： 

蓝灯的开发团队的作者fffw 今天对于我编译提出的问题作出了以下解释，2.0.0-beta11应该是贴错了tag标签，2.0.0-beta11不应该是当前版的标签： 
if you `git log`, you'll see that the tag `2.0.0-beta11` is really old, might be mistakenly tagged.
 Branch `valencia` always contains current production code.

详见：https://github.com/getlantern/lantern/issues/2878

所以，当我从valencia branch再 git checkout 截取2.0.0-beta11就出现了错误，这也是我第一次编译不成功的原因。
事实上valencia branch根本就没有2.0.0-beta11这个分支。

git clone -b valencia --depth 1 --single-branch git://github.com/getlantern/lantern.git
