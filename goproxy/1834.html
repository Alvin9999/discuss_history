除了赛风这个必备软件之外，还需要 Proxifier 这个软件来达到目的，这是商业软件，不过破解版在网上也是比较容易找到的。3.28 版本在 Windows 8/8.1 下可能有问题，锁不住赛风的进程之类的，推荐使用 3.21 版本。
安装并输入注册码激活后，建议重启下操作系统，否则后面的步骤可能不成功。还有，在 Proxifier 中订制规则强制赛风使用 Akamai CDN 服务前，请先运行下赛风，保证它是可以连接上服务器的。我曾经碰到过在 Proxifier 中订制规则前没有运行过赛风，结果赛风根本连不上服务器的情况..

我在MEGA网盘上传了附带注册机的版本

第一步、订制规则（这里就直接借用 warmingf...@gmail.com 的截图了）

规则1使 *.akmaihd.net 直连
https://i.imgur.com/IUvNSgl.jpg

注意如果使用的是 Proxifier 3.28 版本，这里的 Applications 里要填写：psiphon3.exe;psiphon-tunnel-core.exe，否则可能在软件里看不到赛风的进程

规则2使赛风访问其他服务器IP被屏蔽
https://i.imgur.com/K3m3HCa.jpg

要确保规则1在规则2上面，否则就失效了
https://i.imgur.com/gfnYeKz.jpg

并让 Proxifier 处理直连的程序，这样做使网络情况更直观
https://i.imgur.com/nfXu2ab.jpg

小Tips：
1.在 Applications 框里填写时，不需要点 Browse... 按钮找，直接复制粘贴：psiphon3.exe;psiphon-tunnel-core.exe 及 psiphon-tunnel-core.exe 进去就行了
2.订制好规则后，可以保存，并导出为配置文件，卸载重装等情况下再导入即可

保存：File -> Save Profile As...
导出：File -> Export Profile...
导入：File -> Import Profile...

第二步、修改本机 hosts 文件，加一行

210.0.156.15 yryirjrhqffsxwpg-a.akamaihd.net

注意 210.0.156.15 这个IP地址是 warmingf...@gmail.com 给出的，他用这个IP速度很快，你就不一定了，还需要更换IP地址，方法后文会讲。

小Tips：修改 hosts 文件后，是不会立即生效的。想生效，方法有二
1、重启计算机（这...改次 hosts 文件就要重启一次，SSD硬盘还好，要用的机械硬盘，那重启一次得等好长时间啊，所以还有方法2）
2、按 Win + R 键调出“运行”对话框，输入：cmd，回车，在弹出的命令行窗口中，输入：ipconfig /flushdns，回车

对了，在用上述方法2之前，请关闭浏览器，做完再打开，再试通过赛风上网速度是否变快了

这个方法的关键在于 Proxifier 这个软件，这个软件很强大，可以让不自带代理服务器设置的软件通过代理服务器连网，可以让软件只能连接某地址，等等。
说这个方法比较复杂，问题也出在 Proxifier 这个软件上，比如在进程里找不到赛风的 psiphon-tunnel-core.exe，比如造成系统网络问题。我已经碰到过几次因为 Proxifier 这个软件造成无法上网只能重启系统了，所以在用第2种方法成功后就卸载了这个软件。

再来说方法2，这次要用到 3proxy 这个俄罗斯人写的软件。说到这个软件，我以前在用地下铁路时用过它（不知道现在还有多少人记得“地下铁路”了，曾经广发邀请码搞得象传销一样...）

1、下载 warmingf...@gmail.com 已经做好基本设置的 3proxy 软件
https://copy.com/pU0aZGd0oBsAigwS/3proxy.7z
（注意压缩包加密了，否则copy.com会报有病毒不让分享，解压密码：goagent）

我在MEGA网盘上也传了一份，压缩包未加密：
https://mega.co.nz/#!QloC1LLL!8RAcPdwMOGH0SfpBW_De2yoXVYn8-FVE98OIDb5VctE

  1. 解压，编辑 config.txt 这个文件以修改其中的 Akamai IP 地址为在你的网络环境下表现好的 IP 地址，不修改默认是 210.0.156.15
  2. 打开 3proxyLauncher.exe （修改自 Phus Lu 的开源 goagent.exe ）
  3. 打开赛风，做如下操作 https://i.imgur.com/lNo4bnu.jpg
  4. 关闭赛风，重新打开赛风

最后，如果你选用的香港服务器IP，建议在赛风里将服务器改成：Singapore（新加坡），速度会更快点

这个方法简单不少吧，不用改 hosts，系统也不会出现奇怪的网络问题，推荐使用。

那为啥还要方法一呢。方法还是多点比较好吧，而且你喜欢研究的话，Proxifier 还是蛮有用的。

好了，现在还有一个问题，前面说过要找更适合自己的IP地址，怎么找呢，可以用 checkgoogleip 或 gogotester，虽然看软件名字都是找 Google 可用IP的，但也能用来找其它的IP

获取随机IP是每组依次获取随机个数量的，因此一组的IP数越少，越有机会会检查，当然获取随机IP会先排除上次查询失败的IP
