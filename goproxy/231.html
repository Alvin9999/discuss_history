ARwMq9b6 commented Oct 2, 2015

俺这 lantern-*.yaml 里的 IP 变动频繁，写了一个 Windows 批处理文件检测 IP 变化并备份新增的 IP 及其对应的配置文件，用到到第三方工具有 NirCmd ( nircmdc.exe ) 和 cygwin ( ls.exe, grep.exe, cp.exe ) ，乱七八糟的。

check_lantern_yaml.bat

cd /d C:\Users\Administrator\AppData\Roaming\Lantern

:Find_Lantern_Yaml_Name
ls -1|FINDSTR /R /C:"lantern-.*\.yaml" > %~dp0Lantern_Yaml_Name.txt||GOTO NoLanternYamlFile
GOTO Get_Lantern_Yaml_Name

:NoLanternYamlFile
nircmdc infobox "No Lantern yaml file" "Warning!"
GOTO Find_Lantern_Yaml_Name

:Get_Lantern_Yaml_Name
FOR /F %%i IN (%~dp0Lantern_Yaml_Name.txt) DO SET Lantern_Yaml_Name=%%i

:Find_If_Proxyall
findstr /C:"proxyall: true" "%Lantern_Yaml_Name%"||GOTO Proxyall_Is_Flase
GOTO Find_Get_New_IP

:Proxyall_Is_Flase
nircmdc infobox "Please set proxyall to true" "Warning!"
GOTO Find_If_Proxyall

:Find_Get_New_IP
grep -P -o "fallback\-.+[^\:]" "%Lantern_Yaml_Name%">%~dp0ip_new.txt
FOR /F %%A IN (%~dp0ip_new.txt) DO set ip_new=%%A

FINDSTR /C:"%ip_new%" "%~dp0ip_list.txt"||GOTO Found_New_IP
GOTO End

:Found_New_IP
"*********************\Babun\cygwin\bin\echo.exe" -e %ip_new%>>%~dp0ip_list.txt
cp "%Lantern_Yaml_Name%" "****************\Backups\Configurations\%ip_new%_%Lantern_Yaml_Name%"

:End

ARwMq9b6 commented Oct 4, 2015

@alresvor 关于 Transparent Proxy

http://ipaddress.com/ 上的 Transparent Proxy 一项的值和 Header 里的 X-Forwarded-For 有关， Header 如下：

GET / HTTP/1.1
Host: ipaddress.com
User-Agent: Mozilla/5.0 (Macintosh; Intel Mac OS X 10_10_1) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/39.0.2171.65 Safari/537.36
Accept: text/html,application/xhtml+xml,application/xml;q=0.9,image/webp,*/*;q=0.8
Accept-Language: en-US,en;q=0.8
Accept-Encoding: gzip,deflate,sdch
DNT: 1
X-Forwarded-For: 1.1.1.2
Via: 1.1 1.1.1.1
If-None-Match: cnnoa2o
Connection: keep-alive

客户端（如浏览器）中能够自定义这个值。一款 Firefox 扩展 Random Agent Spoofer 能轻松实现自定义或每次访问网页随机一个 X-Forwarded-For IP 地址。
