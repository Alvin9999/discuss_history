cxjava commented Aug 7, 2016 • edited

@distance00 参见以下过程，如有不懂，请留言。

  • 安装heroku toolbelt 客户端 ：Heroku Toolbelt for Windows 里面自带了git和heroku命令，后面步骤会用到。
  • 打开CMD.exe，如果不能登录，设置相应的代理。运行以下命令：
set http_proxy=http://127.0.0.1:1080
set https_proxy=http://127.0.0.1:1080
heroku login
git clone -b server.php-go https://github.com/phuslu/goproxy
cd goproxy
git branch -m server.php-go master
git branch
  • 修改index.go里面的默认密码后，运行以下命令：
git add . -A
git commit -am "upgrade"
  • 获取自己的app，运行以下命令，如果没有app，请运行 heroku apps:create：
heroku apps

结果：

=== My Apps

def-adfd-9117
adf-adfh-2549
adf-aa-6590
bb-cc-9881
  • ee-abc-9754 为你的APP名称，依次运行以下命令就可以把你的app的server端全部更新了:
heroku git:remote -a ee-abc-9754
git push -f heroku master
heroku git:remote -a def-adfd-9117
git push -f heroku master
heroku git:remote -a adf-adfh-2549
git push -f heroku master
heroku git:remote -a adf-aa-6590
git push -f heroku master
heroku git:remote -a bb-cc-9881
git push -f heroku master
