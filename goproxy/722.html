公布一下通过测试的前置黄狗版本为3.0.8、3.1.1、3.1.4。

例如3.0.8版本的配置：
path = /2
profile = google_cn
mode = rtmpe
sslversion = DTLSv12
hosts = [google_hk]hosts
crlf = 1
obfuscate = 1
validate = 1
options = rc4

后置的3.2.3的配置：
path = /_gh/
mode = rtmpe
ipv6 = '0或者1'
sslversion = DTLSv12
window = 7
cachesock = 1
headfirst = 1
keepalive = 1
obfuscate = 1
validate = 1
transport = 0
options = rc4
regions =
