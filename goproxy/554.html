phuslu commented Apr 18, 2016 • edited

r400 删掉了 EnableHTTP2 选项。变成了

  • DisableHTTP2 完全禁用 http2, 永远使用 http1
  • ForceHTTP2 强制使用 http2, 这个遇到不支持的 IP 就有 Unexcept EOF 错误了。

这样我觉得比较满意了。不过对于 IP 数目较少或者纯 IPv6 的配置，我建议还是打开 ForceHTTP2 这样可以确保不会被回退到 http1 模式的 IP 拖慢速度。

现在默认的行为是对所有 IP 都尝试使用 http2 协议，不支持的话就对此 IP 回退到 http1 协议。比较灵活。
