1、手机安装termux
2、打开运行termux 输入 uname -a 获得适应你手机版本的goproxy(①、armv7l/armv6l 对应 goproxy_linux_arm 或 goproxy_linux_arm_cgo

②、aarch64 对应 goproxy_linux_arm64)
3、下载goproxy(如我的手机是aarch64下载goproxy_linux_arm64-r1174.tar.xz)放到手机根目录并改名为goproxy.tar.xz(改名是为了不让文件名太长，你也可以不该)
4、cd /sdcard
5、tar -Jxvf goproxy.tar.xz
6、将你的gae.user.json放到解压后得到的goproxy文件夹地下(解压后goproxy文件夹在手机内存卡根目录下)
7、cp -r /sdcard/goproxy/. /data/data/com.termux/files/usr/tmp/goproxy
8、chmod -R 777 /data/data/com.termux/files/usr/tmp/goproxy
9、cd /data/data/com.termux/files/usr/tmp/goproxy
10、./goproxy

步骤3-8注释说明：
4、转到手机内存卡位置
5、解压goproxy
7、复制goproxy 到/data/data/com.termux/files/usr/tmp
8、获取/data/data/com.termux/files/usr/tmp/goproxy的文件权限(没权限无法运行)
9、转到/data/data/com.termux/files/usr/tmp/goproxy文件夹下
10、运行goproxy

============
11、手机上设置使用
①手机wifi上设置http://127.0.0.1:8087/proxy.pac或设置代理服务器：127.0.0.1，端口：8087
②移动网络使用设置：接入点名称(APN)—>中国移动(china mobile）GPRS —>代理服务器：127.0.0.1，端口：8087

12、证书安装
①使用命令复制(导出到内存卡)：
A、cd /data/data/com.termux/files/usr/tmp/goproxy
B、cp GoProxy.crt /sdcard
②使用浏览器地址下载： http://127.0.0.1:8087/
③手机设置—>更多设置—>系统安全—>从存储设备安装—>选择证书安装
④将证书变为系统级：#1628

13、gp配合Firefox浏览器使用(注：Firefox浏览器配合pan插件使用后可以不用设置wifi和移动网络的服务器)：
①安装Firefox浏览器
②安装pan插件
③Firefox浏览器安装证书(注：Firefox使用的是本身的证书)：
A、从手机内存安装：如把证书放到手机根目录，然后在firefox地址栏输入 /sdcard/goproxy.crt回车
B、直接在地址栏输入 http://127.0.0.1:8087 ，后选择证书GoProxy.crt导入证书

注：三个选项均需打钩
④Pan插件设置：
地址栏输入 about:config —>搜索defaultProxy —>将默认的SS改为GoAgent

14、termux无法使用更新脚本解决方法：
解决termux本身没有wget和curl命令，无法使用脚本更新及下载文件的问题。
①apt-get update
②apt-get install curl ，后提示提示安装与否，输入 y
③apt-get install wget，后提示提示安装与否，输入 y
15、利用termux:widget 建立goproxy脚本快捷方式 ：#1475

手机上纯在证书问题目前无解，请使用浏览器上外网

步骤2、4、5、7、8、9、10在termux上运行(提示：在termux长按可复制粘贴本教程中命令，节省输入命令的时间)
